# Data Types

###### *1. Identify the data type or class for each of the following values:*

| **Value**         | **Type**   |
| ----------------- | ---------- |
| `'True'`          | String     |
| `False`           | Boolean    |
| `(1, 2, 3)`       | Tuple      |
| `1.5`             | Float      |
| `[1, 2, 3]`       | List       |
| `2`               | Integer    |
| `range(5)`        | Range      |
| `{1, 2, 3}`       | Set        |
| `None`            | NoneType   |
| `{'foo' : 'bar'}` | Dictionary |

###### *2. Create a tuple called `names` that contains several pet names. For instance:*

| **Name**     |

| Neptune      |
| Darwin       |

```python
names = (

```

###### *3. Create a dictionary named `pets` that contains a list of pet names and the type of animal. For instance:*

| **Name**     | **Animal** |
| ------------ | ---------- |
| Asta         | dog        |
| Butterscotch | cat        |
| Pudding      | cat        |
| Neptune      | fish       |
| Darwin       | lizard     |

```python
pets = {
    'Asta' :         'dog',
    'Butterscotch' : 'cat',
    'Pudding' :      'cat',
    'Neptune' :      'fish',
    'Darwin' :       'lizard',
}
```
